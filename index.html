Cara Menghubungkan ke Spreadsheet:

Setiap kelas otomatis dibuatkan Sheet terpisah. Rekap baru akan bergeser ke kanan (kolom baru) agar rapi dan tidak menumpuk!

  1. Buat Google Spreadsheet baru.
  2. Buka menu Ekstensi > Apps Script.
  3. Hapus semua kode bawaan, lalu salin-tempel kode Apps Script di bawah ini.
  4. Klik Terapkan (Deploy) > Penerapan Baru.
  5. Pilih tipe Aplikasi Web, setel akses ke "Siapa saja (Anyone)".
  6. Salin URL Aplikasi Web yang dihasilkan dan tempel ke baris variabel CONFIG_WEBHOOK_URL di dalam kode HTML aplikasi ini.
